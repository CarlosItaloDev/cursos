Supondo que esse segundo texto não teria relação com a imagem, vamos desvinculá-lo dela com forma de alinhar diferente da outra, através da tag style atributo class no head com atributo class e clear.

Temos aqui uma nova imagem utilizando o atributo class, porém sem o class seguido de clear utilizados no parágrafo do exemplo anterior, vinculando assim o texto a imagem

Outra tag interessante é a tag figure seguida pela figcaption, essas tags tem o objetivo criar uma legenda para imagem,normalmente é utilizada para dar créditos a um fotógrafo ou artista.É interessante observar quebra que a simples inclusão da figure mesmo utilizando a mesma class da bandeira anterior, altera o posicionamento da imagem.

Utilizando link

Vamos agora trabalhar com link a seguir, pelo próprio texto você será direcionado a uma página, desta forma utilizando a tag a href="endereço", seja esse endereço para a pasta do pŕoprio projeto para um arquivo html ou um endereço da web, como esse exemplo à página google.

Em ambos exemplos acima, quando o usuário clica no link é fechada a página para que abra a do endereçamento, para que isso não ocorra, devemos usar a tag a com os seguintes atributos href="endereço" target= "blank" como verificará no clique aqui.

Links relativos

Nos exemplos acima temos um link para uma página na pasta do projeto e outra para endereço de web, porém é possível fazer referência a essa pasta pai incluindo ../ antes do endereço do projeto ou avô incluindo ../../, isso dentro da tag a href="",segue o link de uma página que contem esse recurso arquivo filho.

Link para própria página

Geralmente é utilizado no início da página html, como foi nesta,levando o usuário a navegar pelos demais campos da página. Isso permite encontrar o tema de interesse de forma rápida. É utilizado principalmente em páginas de conteúdos extensos. veja em clique aqui.

Para fazer isso basta utilizar o atributo id="palavra do menu" dentro da tag H no decorrer da página e colocar tag a com href="#palavra do menu", desta forma o conteúdo seerá exibido e o cabeçalho virá para o topo da página se seu conteúdo for superior ao tamanho da tela. Para retornar ao topo, basta colocar o #.
