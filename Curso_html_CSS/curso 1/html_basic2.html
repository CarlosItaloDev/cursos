Fonte no html

Para mudança de estilo de fonte, tamanho e cores, no html é mais comum a utilização do CSS atravrés da tag style, porém o html possui campo para tal finalildade, apesar de estar em desuso que é a tag fonte face="nome da fonte" size="tamanho" color="codigo da cor ou nome".

Existe uma ferramenta para gerar códigos de cores (color picker) em grande quantidade clique aqui a partir de suas próprias imagens.

Inserindo imagem

Para inserir uma imagem devemos utilizar a tag img depois o atributo src="endereço da imagem".

É possível controlar o tamanho da imagem com os atributos width para largura e height para altura, se colocar apenas um dos atributos, a imagem altera, mas mantendo as proporções.

Para saber o real tamanho da imagem é só ir a pasta destinada a imagens do projeto e ver a propriedade, nos atributos height e width não é preciso colocar px após o número, pois no html5 esse é padrão.

Arquivo index no html

No html o arquivo de nome index.html significa que é a página principal,mas pode ser encontrada também com o nome de default.html, portanto esta que abrirá primeiro pelo browser, é importante ressaltar que que o index torna o sistema mais seguro a invasões, pois outro nome, será exibida as opções de abertura para o usuário, permitindo que este tenha acesso a pasta de conteúdos do servidor.

Mapeamento de imagem

O mapeamento de imagem ocorre quando existe a necessidade de uma mesma imagem ter links para mais de uma página, diferente do link por imagem em que a tag image é aberta entre a tag anchor.

Podemos criar dois tipos de mapeamento de imagem:

  1. sever-side (servidor)
      O tipo de mapeamento do lado do servidor é utilizado com um arquivo de definição das coordenadas.
  2. client-side (cliente)
      O mapeamento feito do lado do cliente ocorre no documento html.
    • O mapeamento ocorre utilizando formas geometricas que possuem coordenadas particulares dentro de um plano cartesiano (x,y)
      1. Retângulo coords= "x1,y1,x2,y2"
      2. circulo coords= "x,y,r"
      3. polígono coords= "x1,y1,x2,y2,x3,y3..."
    • O mapeamento ocorre ao colocar o atributo, vejamos este exemplo a seguir com indentação: usemap="#nome_do_mapeamento": 
                                      img src="imagem_menu.gif" usemap="#menu" border="0">

	                            map name="menu"
		                            area shape="rect" coords="11,13,115,52" href="http://www.fb.org.br" alt="Item 1"
		                            area shape="rect" coords="11,55,114,94" href="http://www.ev.org.br" alt="Item 2"
                                 map
      Nesse caso, o shape foi "rect" para retângulo, mas pode ser "circle" e "polygon" a depender das coordenadas.
